# Favicon

This directory contains the favicon for MD Viewer.

- favicon.svg - Main favicon (SVG format, supported by modern browsers)

## Browser Support

Modern browsers support SVG favicons:
- Chrome 80+
- Firefox 41+
- Safari 15.4+
- Edge 80+

For older browsers, you can convert the SVG to PNG/ICO using online tools.

## Usage

The favicon is referenced in HTML files:
```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
```
